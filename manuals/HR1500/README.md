# HR1500 Documentation

Documentação técnica do sistema HR1500.

## Objetivo

Este repositório contém toda a documentação necessária para instalação, configuração, operação e manutenção do sistema HR1500 executando Ubuntu Server LTS em computadores x86-64.

O objetivo é permitir que qualquer técnico consiga instalar um equipamento novo utilizando apenas este manual.

## Conteúdo

- Instalação do Ubuntu Server LTS
- Configuração da BIOS
- Configuração de rede
- Configuração do SSH
- Instalação da aplicação
- Configuração do systemd
- Operação Headless
- Console Serial
- Clonagem de discos
- Atualização de software
- Troubleshooting
